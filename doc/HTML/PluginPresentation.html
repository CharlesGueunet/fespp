How to Load a file ?


Pour charger le plugin sous ParaView il faut sélectionner le menu « Tools ».
Figure 1. ParaView menu.

A partir de ce menu lancer le Plugin Manager (Manage Plugins…).
Figure 2. Plugin Manager link.

Le plugin manager s’ouvre.
Sélectionner « Load New… » et parcourer l’arborescence de votre système pour sélectionner le plug-in.
Figure 3. Plugin Manager.

Une fois le plug-in chargé, il doit apparaitre dans la liste du « plugin manager » comme ci-dessous :
Figure 4. Plugin loaded.

Fermer le « plugin manager ».
2 nouvelles fenêtres apparaissent :
  • MetaData Panel
    Figure 5. MetaData Panel.

  • Selection Panel
    Figure 5. Selection Panel.

Ces 2 fenêtres sont positionnables comme vous le voulez.
